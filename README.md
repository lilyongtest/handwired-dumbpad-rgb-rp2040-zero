## Status
- Prototype

## Features
- Single-controller split keyboard design
- Left and right halves connected via LAN (RJ45) cable
- Uses the 8 wires of an Ethernet cable for interconnection
- Duplex Matrix design to reduce required GPIO pins
- Efficient pin usage for compact controller configurations

## Specifications

| Item | Description |
|-----|-----|
| Layout | Column-staggered |
| Keys | 52 keys |
| Controller | RP2040-Zero |
| Firmware | QMK |
| Connection | USB-C |
| Case | cardboard / lether  |
